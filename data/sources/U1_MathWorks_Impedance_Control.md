# Advanced Robotic Manipulation with Impedance Control

* **ID:** U1
* **Source:** MathWorks Technical Article
* **URL:** https://www.mathworks.com/company/technical-articles/enhancing-robot-precision-and-safety-with-impedance-control.html

## Summary and Conclusions
This article discusses enhancing robot precision and safety using impedance control. It provides an intuitive explanation of how impedance control allows robots to behave like virtual spring-damper systems, regulating the interaction force with the environment by adjusting stiffness and damping dynamically. This is crucial for medical robotics (like ultrasound scanning) where the robot must interact safely with soft human tissues.
