# Admittance Control in Robotics

* **ID:** U2
* **Source:** Emergent Mind
* **URL:** https://www.emergentmind.com/topics/admittance-control

## Summary and Conclusions
This topic overview describes admittance control in robotics. It covers the second-order Mass-Spring-Damper (MSD) virtual model: M_d * x_ddot + B_d * x_dot + K_d * x = F_ext. An admittance controller takes external force measurements (from an F/T sensor) and calculates the corresponding reference displacement/velocity for the robot's position controllers, making it highly suitable for industrial robots with stiff position-controlled joints.
